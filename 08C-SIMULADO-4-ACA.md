# ✅ GABARITO COMENTADO

## Português
1. **B** — o texto diz "substituir a suposição pelo fato".
2. **B** — na era digital, o método mais honesto é o mais antigo → aparente contradição = paradoxo/paradoxalmente.
3. **B** — trocar estimativas por dados verificados.
4. **B** — "melhores" é o comparativo correto; "mais melhores/bons/ótimos" são pleonasmos viciosos.
5. **A** — "Vinte e uma candidatas foram aprovadas": numeral composto terminado em "uma" concorda no feminino com o substantivo feminino plural.
6. **C** — "Informe-me **sobre** tudo" é a regência padrão de "informar" no sentido de comunicar (informar algo a alguém / informar sobre algo).
7. **B** — identifica/restringe "a equipe" → adjetiva **restritiva** (sem vírgulas).
8. **B** — "mau momento" (adjetivo = ruim); os demais empregam "mal/mau" incorretamente em alguns contextos.
9. **B** — "silêncio gritou" une ideias opostas → **paradoxo/oxímoro**.
10. **B** — "senão" = "a não ser"; "se não" introduz condição. "Não fez outra coisa **senão** reclamar" está correto.
11. **B** — "fosse" é o pretérito imperfeito do **subjuntivo** (modo subjuntivo, tempo passado).
12. **B** — "referiu" rege "a"; "à qual" = crase antes do pronome relativo "a qual" (fem.).
13. **B** — passiva analítica: "Os dados **foram publicados** pelo IBGE".
14. **C** — após o Acordo de 2009, "pára" (verbo parar) **perdeu** o acento; "pôde" (passado de poder) mantém.
15. **B** — explica/detalha "o processo" → **aposto**.
