# 🟢 NÍVEL FÁCIL (20)

**INFO-F01.** O sistema operacional é responsável por:
A) redigir documentos. B) gerenciar o hardware e os recursos do computador, sendo a interface entre software e hardware. C) navegar na internet. D) criar planilhas. E) enviar e-mails.
> ✔️ **B.** O SO gerencia hardware e serve de plataforma para os demais softwares.

**INFO-F02.** No Windows, o atalho de teclado para **copiar** um texto selecionado é:
A) Ctrl+X B) Ctrl+V C) Ctrl+C D) Ctrl+Z E) Ctrl+A
> ✔️ **C.** Ctrl+C = copiar; Ctrl+X = recortar; Ctrl+V = colar; Ctrl+Z = desfazer.

**INFO-F03.** O atalho **Ctrl+Z** serve para:
A) refazer. B) desfazer a última ação. C) salvar. D) fechar. E) selecionar tudo.
> ✔️ **B.** Ctrl+Z = **desfazer** (undo).

**INFO-F04.** A extensão de arquivo **.docx** é associada a:
A) planilhas Excel. B) apresentações PowerPoint. C) documentos Word. D) imagens. E) arquivos compactados.
> ✔️ **C.** .docx = Microsoft Word (documentos de texto).

**INFO-F05.** A extensão **.xlsx** é associada a:
A) Word. B) Excel. C) PowerPoint. D) PDF. E) Notepad.
> ✔️ **B.** .xlsx = Microsoft Excel (planilhas).

**INFO-F06.** A extensão **.pptx** é associada a:
A) Word. B) Excel. C) PowerPoint. D) Access. E) Outlook.
> ✔️ **C.** .pptx = Microsoft PowerPoint (apresentações).

**INFO-F07.** O navegador de internet é um software usado para:
A) editar imagens. B) acessar páginas na web. C) criar planilhas. D) gerenciar arquivos. E) enviar fax.
> ✔️ **B.** Navegador (browser) = acesso a páginas web.

**INFO-F08.** O protocolo usado para enviar e-mails é o:
A) HTTP B) FTP C) SMTP D) SSH E) DNS
> ✔️ **C.** SMTP (Simple Mail Transfer Protocol) = envio de e-mails.

**INFO-F09.** RAM significa:
A) Registro de Acesso à Memória. B) Random Access Memory (memória de acesso aleatório). C) Rede de Armazenamento em Massa. D) Recurso Administrativo de Memória. E) Repositório Automático de Mídia.
> ✔️ **B.** RAM = memória de acesso aleatório (volátil, usada durante a execução dos programas).

**INFO-F10.** A diferença entre a memória **RAM** e o **HD/SSD** é que a RAM é:
A) permanente; o HD é temporário. B) temporária (volátil) e mais rápida; o HD é permanente e mais lento. C) usada para armazenamento permanente. D) exclusiva para o sistema operacional. E) independente do processador.
> ✔️ **B.** RAM = volátil e rápida; HD/SSD = persistente e mais lento.

**INFO-F11.** O antivírus serve para:
A) acelerar o computador. B) detectar, bloquear e remover softwares maliciosos. C) criar backups automáticos. D) gerenciar senhas. E) compactar arquivos.
> ✔️ **B.** Antivírus = proteção contra malwares.

**INFO-F12.** Phishing é uma técnica de ataque que:
A) destrói fisicamente o hardware. B) engana o usuário para que revele dados pessoais ou clique em links maliciosos. C) sobrecarrega servidores. D) criptografa arquivos para resgate. E) intercepta sinais de Wi-Fi.
> ✔️ **B.** Phishing = engenharia social para roubo de dados.

**INFO-F13.** O formato de arquivo **PDF** foi criado para:
A) edição de texto. B) criação de planilhas. C) apresentação de slides. D) compartilhar documentos de forma fiel, independente do sistema ou software. E) armazenar bancos de dados.
> ✔️ **D.** PDF (Portable Document Format) = portabilidade e fidelidade visual.

**INFO-F14.** A lixeira do Windows serve para:
A) excluir arquivos permanentemente de imediato. B) armazenar arquivos excluídos temporariamente, permitindo recuperação. C) compactar arquivos. D) realizar backups. E) gerenciar a memória RAM.
> ✔️ **B.** A lixeira guarda arquivos deletados até o esvaziamento.

**INFO-F15.** O atalho **Ctrl+S** serve para:
A) selecionar tudo. B) salvar o arquivo atual. C) fechar o programa. D) abrir novo documento. E) imprimir.
> ✔️ **B.** Ctrl+S = **salvar** (Save).

**INFO-F16.** Wi-Fi é uma tecnologia de:
A) armazenamento em nuvem. B) conexão sem fio à rede (wireless). C) transmissão de dados por cabo. D) impressão sem fio. E) backup automático.
> ✔️ **B.** Wi-Fi = conexão sem fio (wireless) à rede.

**INFO-F17.** No Windows Explorer (Explorador de Arquivos), a tecla **F2** serve para:
A) abrir o arquivo. B) renomear o arquivo ou pasta selecionado. C) excluir. D) copiar. E) criar nova pasta.
> ✔️ **B.** F2 = renomear no Windows Explorer.

**INFO-F18.** O endereço de e-mail é composto por:
A) nome de usuário, arroba (@) e domínio. B) protocolo, IP e porta. C) login, senha e servidor. D) nome, CPF e e-mail. E) usuário, senha e URL.
> ✔️ **A.** Formato: usuario@dominio.com.

**INFO-F19.** No contexto de segurança digital, uma senha **forte** deve conter:
A) apenas letras minúsculas. B) apenas números. C) combinação de letras maiúsculas, minúsculas, números e símbolos. D) o nome do usuário. E) menos de 6 caracteres.
> ✔️ **C.** Senha forte = combinação variada de caracteres.

**INFO-F20.** A função do **firewall** é:
A) acelerar a conexão à internet. B) monitorar e controlar o tráfego de rede, bloqueando acessos não autorizados. C) criar backups. D) compactar dados. E) gerenciar impressoras.
> ✔️ **B.** Firewall = barreira de controle de tráfego de rede.

# 🟡 NÍVEL MÉDIO (20)

**INFO-M01.** No Microsoft Word, a formatação **negrito** é ativada pelo atalho:
A) Ctrl+I B) Ctrl+B C) Ctrl+N D) Ctrl+U E) Ctrl+E
> ✔️ **C.** No Word em português, **Ctrl+N** = negrito (Bold); Ctrl+I = itálico; Ctrl+S = sublinhado.

**INFO-M02.** No Excel, a fórmula que **soma** um intervalo de células A1 até A10 é:
A) =TOTAL(A1:A10) B) =SOMA(A1:A10) C) =SOMAR(A1:A10) D) =ADD(A1:A10) E) =SUM(A1,A10)
> ✔️ **B.** =SOMA(A1:A10) — função nativa do Excel em português.

**INFO-M03.** No Excel, a fórmula **=MÉDIA(B1:B5)** calcula:
A) a soma dos valores. B) o maior valor. C) a média aritmética dos valores do intervalo. D) o menor valor. E) o total de células preenchidas.
> ✔️ **C.** =MÉDIA calcula a **média aritmética**.

**INFO-M04.** No Excel, o símbolo **$** numa referência de célula (ex.: $A$1) indica:
A) valor monetário. B) referência relativa. C) referência absoluta (não muda ao copiar a fórmula). D) célula vazia. E) fórmula de texto.
> ✔️ **C.** $ fixa a referência → **absoluta**.

**INFO-M05.** No PowerPoint, o modo de exibição que mostra todos os slides em miniatura para reordenar é o:
A) Normal. B) Leitura. C) Classificação de Slides. D) Apresentação de Slides. E) Anotações.
> ✔️ **C.** **Classificação de Slides** exibe miniaturas para reorganizar.

**INFO-M06.** O protocolo **HTTPS** difere do HTTP por:
A) ser mais lento. B) usar criptografia (SSL/TLS) para proteger a comunicação. C) funcionar apenas em redes locais. D) não suportar imagens. E) ser exclusivo de e-mail.
> ✔️ **B.** HTTPS = HTTP + **criptografia** (SSL/TLS).

**INFO-M07.** Um **backup** deve ser realizado para:
A) acelerar o computador. B) proteger dados contra perda por falha, exclusão acidental ou ataque. C) aumentar a memória RAM. D) melhorar a conexão à internet. E) reduzir o consumo de energia.
> ✔️ **B.** Backup = cópia de segurança dos dados.

**INFO-M08.** O protocolo **FTP** é utilizado para:
A) envio de e-mails. B) transferência de arquivos entre computadores. C) navegação segura. D) resolução de nomes de domínio. E) videoconferência.
> ✔️ **B.** FTP (File Transfer Protocol) = transferência de arquivos.

**INFO-M09.** O endereço **IP** serve para:
A) identificar o fabricante do computador. B) identificar unicamente um dispositivo em uma rede. C) criptografar dados. D) armazenar senhas. E) nomear arquivos.
> ✔️ **B.** IP = identificador único de dispositivo na rede.

**INFO-M10.** Um **vírus de computador** se diferencia de um **worm** porque o vírus:
A) se propaga sozinho sem precisar de arquivo hospedeiro. B) precisa de um arquivo hospedeiro para se propagar. C) não causa danos. D) é detectado antes de se instalar. E) só ataca dispositivos móveis.
> ✔️ **B.** Vírus precisa de hospedeiro; **worm** se propaga autonomamente pela rede.

**INFO-M11.** O **ransomware** é um tipo de malware que:
A) espiona o usuário silenciosamente. B) exibe propagandas indesejadas. C) criptografa os arquivos da vítima e exige resgate para liberá-los. D) rouba senhas digitadas. E) desativa o antivírus.
> ✔️ **C.** Ransomware = sequestro de dados mediante resgate.

**INFO-M12.** A **nuvem (cloud computing)** permite:
A) apenas armazenar arquivos localmente. B) acessar serviços, armazenamento e processamento via internet, de qualquer dispositivo. C) dispensar a conexão à internet. D) aumentar a velocidade do hardware local. E) substituir o sistema operacional.
> ✔️ **B.** Cloud = serviços e armazenamento via internet, acessíveis remotamente.

**INFO-M13.** No Windows, o atalho **Alt+F4** serve para:
A) abrir o explorador. B) fechar a janela/aplicativo ativo. C) salvar. D) imprimir. E) minimizar.
> ✔️ **B.** Alt+F4 = **fechar** a janela ativa.

**INFO-M14.** O **DNS** (Domain Name System) tem a função de:
A) criptografar e-mails. B) traduzir nomes de domínio (ex.: www.ibge.gov.br) em endereços IP. C) gerenciar senhas. D) armazenar cookies. E) bloquear vírus.
> ✔️ **B.** DNS = "lista telefônica" da internet (nome → IP).

**INFO-M15.** No Excel, a fórmula **=SE(A1>10;"Aprovado";"Reprovado")** retorna "Aprovado" quando:
A) A1 é menor que 10. B) A1 é igual a 10. C) A1 é maior que 10. D) A1 está vazia. E) A1 contém texto.
> ✔️ **C.** A condição é A1 **>** 10.

**INFO-M16.** O **Bluetooth** é uma tecnologia de:
A) armazenamento em nuvem. B) comunicação sem fio de curto alcance entre dispositivos. C) conexão via cabo de rede. D) transmissão de dados por satélite. E) identificação biométrica.
> ✔️ **B.** Bluetooth = comunicação sem fio de **curto alcance**.

**INFO-M17.** O conceito de **licença de software** define:
A) o preço do software. B) os termos de uso e direitos do usuário sobre o software. C) a qualidade do software. D) o suporte técnico. E) a velocidade de instalação.
> ✔️ **B.** Licença = termos de uso e direitos do usuário.

**INFO-M18.** O software **livre** (open source) permite:
A) uso gratuito apenas. B) uso, cópia, modificação e redistribuição do código-fonte. C) uso apenas por governos. D) uso sem conexão à internet. E) uso sem necessidade de instalação.
> ✔️ **B.** Software livre/open source = acesso ao código-fonte + liberdade de uso/modificação.

**INFO-M19.** No contexto de redes, **LAN** refere-se a:
A) rede de longa distância. B) rede local (Local Area Network), como a de um escritório. C) rede sem fio. D) rede via satélite. E) rede pública de internet.
> ✔️ **B.** LAN = rede local (escritório, casa); WAN = longa distância.

**INFO-M20.** No Word, o recurso **"Controle de Alterações"** serve para:
A) verificar ortografia. B) registrar as modificações feitas no documento, permitindo aceitar ou rejeitar cada alteração. C) formatar o texto. D) inserir tabelas. E) criar sumário automático.
> ✔️ **B.** Controle de Alterações = rastreamento de edições (útil em revisões colaborativas).

# 🔴 NÍVEL DIFÍCIL (20)

**INFO-D01.** No Excel, a fórmula **=PROCV(A2;$C$1:$D$10;2;FALSO)** busca o valor de A2:
A) na segunda linha do intervalo. B) na primeira coluna de C1:D10 e retorna o valor da segunda coluna da mesma linha encontrada. C) na segunda coluna e retorna a primeira. D) na última linha do intervalo. E) em toda a planilha.
> ✔️ **B.** PROCV: busca o valor na 1ª coluna do intervalo e retorna o da coluna indicada (2ª); FALSO = correspondência exata.

**INFO-D02.** No Excel, a diferença entre **referência relativa** (A1), **absoluta** ($A$1) e **mista** ($A1 ou A$1) é:
A) apenas estética. B) ao copiar a fórmula, a relativa ajusta linha e coluna; a absoluta não ajusta nenhuma; a mista fixa apenas uma das dimensões. C) a absoluta sempre ajusta. D) a mista nunca ajusta. E) só a relativa é usada em PROCV.
> ✔️ **B.** Relativa = ajusta tudo; absoluta = fixa tudo; mista = fixa linha OU coluna.

**INFO-D03.** O protocolo **IMAP** difere do **POP3** no recebimento de e-mails porque o IMAP:
A) baixa e remove o e-mail do servidor. B) mantém os e-mails no servidor, sincronizando em múltiplos dispositivos. C) é usado apenas para envio. D) não suporta anexos. E) criptografa automaticamente todos os e-mails.
> ✔️ **B.** IMAP = e-mails ficam no servidor (sincronizado); POP3 = baixa e remove do servidor.

**INFO-D04.** A criptografia **assimétrica** usa:
A) uma única chave para cifrar e decifrar. B) um par de chaves: pública (para cifrar) e privada (para decifrar). C) nenhuma chave. D) apenas hardware dedicado. E) a senha do usuário como chave.
> ✔️ **B.** Assimétrica = par de chaves pública/privada (ex.: SSL/TLS, assinatura digital).

**INFO-D05.** A diferença entre **vírus, worm, trojan e spyware** é:
A) são sinônimos. B) vírus (hospedeiro), worm (autônomo), trojan (disfarçado de legítimo), spyware (espionagem). C) apenas o vírus causa dano. D) spyware é o mais perigoso sempre. E) trojan se propaga sozinho como worm.
> ✔️ **B.** Cada malware tem mecanismo e objetivo distintos.

**INFO-D06.** No contexto de segurança, **autenticação multifator (MFA)** combina:
A) múltiplas senhas iguais. B) algo que você sabe (senha) + algo que você tem (token/celular) + algo que você é (biometria). C) apenas biometria. D) apenas senha e login. E) múltiplos antivírus.
> ✔️ **B.** MFA = múltiplos fatores de autenticação (sabe + tem + é).

**INFO-D07.** No Windows, o **Gerenciador de Tarefas** (Ctrl+Shift+Esc) permite:
A) criar novos usuários. B) monitorar processos, desempenho e uso de CPU/memória, além de encerrar processos. C) formatar o HD. D) instalar drivers. E) configurar a rede.
> ✔️ **B.** Gerenciador de Tarefas = monitoramento e encerramento de processos.

**INFO-D08.** No Word, o **sumário automático** é gerado a partir de:
A) texto em negrito. B) estilos de parágrafo (Título 1, Título 2…) aplicados aos títulos do documento. C) tabelas inseridas. D) cabeçalho e rodapé. E) número de páginas.
> ✔️ **B.** Sumário automático usa os **estilos de título** (Heading 1, 2…).

**INFO-D09.** A diferença entre **HD (disco rígido)** e **SSD (solid state drive)** é que o SSD:
A) é mais lento e mais barato. B) usa partes mecânicas em movimento. C) é mais rápido, silencioso e resistente a impactos, pois não tem partes móveis. D) tem maior capacidade sempre. E) só funciona como armazenamento externo.
> ✔️ **C.** SSD = memória flash, sem partes móveis → mais rápido e resistente.

**INFO-D10.** O conceito de **VPN (Virtual Private Network)** é:
A) uma rede local física. B) uma rede pública acelerada. C) um túnel criptografado que permite acesso seguro a redes remotas pela internet. D) um protocolo de e-mail. E) um tipo de firewall.
> ✔️ **C.** VPN = túnel criptografado para acesso seguro remoto.

**INFO-D11.** No Excel, a fórmula **=CONT.SE(A1:A20;">"&B1)** conta:
A) todos os valores iguais a B1. B) todos os valores do intervalo maiores que o valor em B1. C) apenas os valores em branco. D) apenas os valores negativos. E) a soma dos valores maiores que B1.
> ✔️ **B.** CONT.SE conta células que atendem ao critério (> valor de B1).

**INFO-D12.** A diferença entre **software legado** e **software atualizado** no contexto de segurança é que o legado:
A) é mais seguro por ser antigo. B) pode ter vulnerabilidades não corrigidas, pois não recebe mais atualizações do fabricante. C) é gratuito. D) tem melhor desempenho. E) é imune a vírus.
> ✔️ **B.** Software sem suporte = vulnerabilidades abertas sem correção.

**INFO-D13.** No PowerPoint, o recurso **"Padrão de Slides"** (Slide Master) serve para:
A) criar uma apresentação do zero. B) definir o layout e a formatação padrão aplicados a todos os slides de uma vez. C) exportar para PDF. D) inserir animações. E) adicionar transições.
> ✔️ **B.** Slide Master = template global de todos os slides.

**INFO-D14.** O conceito de **latência** em redes refere-se a:
A) a velocidade máxima de download. B) o tempo que um pacote de dados leva para ir da origem ao destino. C) a capacidade de armazenamento. D) o número de dispositivos conectados. E) a frequência do sinal Wi-Fi.
> ✔️ **B.** Latência = atraso na transmissão de dados (medido em ms).

**INFO-D15.** No contexto de banco de dados, a linguagem **SQL** é usada para:
A) criar páginas web. B) consultar, inserir, atualizar e excluir dados em bancos de dados relacionais. C) editar imagens. D) programar sistemas operacionais. E) configurar redes.
> ✔️ **B.** SQL (Structured Query Language) = linguagem de bancos de dados relacionais.

**INFO-D16.** A diferença entre **malware** e **exploit** é:
A) são sinônimos. B) malware é software malicioso em geral; exploit é código que aproveita uma vulnerabilidade específica para comprometer o sistema. C) exploit é inofensivo. D) malware sempre exige interação do usuário. E) exploit é um tipo de antivírus.
> ✔️ **B.** Malware = categoria geral; exploit = ataque a vulnerabilidade específica.

**INFO-D17.** No Windows, a diferença entre **"Excluir"** e **"Excluir permanentemente"** (Shift+Delete) é:
A) nenhuma. B) "Excluir" envia à lixeira (recuperável); Shift+Delete remove sem passar pela lixeira (irrecuperável pelo método comum). C) Shift+Delete envia à lixeira. D) "Excluir" é permanente. E) ambos permitem recuperação.
> ✔️ **B.** Shift+Delete = exclusão permanente, sem lixeira.

**INFO-D18.** A **assinatura digital** em documentos eletrônicos garante:
A) apenas a formatação do documento. B) autenticidade (identidade do signatário) e integridade (documento não alterado). C) apenas a confidencialidade. D) o backup automático. E) a impressão fiel.
> ✔️ **B.** Assinatura digital = autenticidade + integridade.

**INFO-D19.** O protocolo **SSH** é utilizado para:
A) navegar em sites seguros. B) transferir arquivos FTP com criptografia. C) acessar remotamente servidores de forma segura e criptografada. D) enviar e-mails. E) resolver nomes de domínio.
> ✔️ **C.** SSH (Secure Shell) = acesso remoto seguro a servidores.

**INFO-D20.** No contexto do Censo, o **DMC (Dispositivo Móvel de Coleta)** exige do ACI conhecimentos de:
A) apenas hardware de impressoras. B) configuração, suporte, manutenção básica e resolução de problemas do dispositivo e do aplicativo de coleta. C) apenas redes Wi-Fi domésticas. D) programação avançada. E) instalação de sistemas operacionais de desktop.
> ✔️ **B.** O ACI apoia o AOR em questões de TI do posto, incluindo o DMC e o sistema de coleta.
