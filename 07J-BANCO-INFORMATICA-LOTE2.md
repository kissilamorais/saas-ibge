# 🟢 NÍVEL FÁCIL (22)

**INFO2-F01.** A tecla **Delete** no teclado serve para:
A) fechar o programa. B) excluir o caractere à direita do cursor (ou o item selecionado). C) salvar o arquivo. D) abrir o menu. E) copiar.
> ✔️ **B.** Delete = exclui à **direita** do cursor (Backspace exclui à esquerda).

**INFO2-F02.** O atalho **Ctrl+A** seleciona:
A) apenas a linha atual. B) todo o conteúdo do documento/janela ativa. C) a última palavra. D) um parágrafo. E) o cabeçalho.
> ✔️ **B.** Ctrl+A = **selecionar tudo** (Select All).

**INFO2-F03.** O atalho **Ctrl+P** abre:
A) a janela de impressão. B) o painel de controle. C) as propriedades do arquivo. D) a área de transferência. E) o gerenciador de tarefas.
> ✔️ **A.** Ctrl+P = **imprimir** (Print).

**INFO2-F04.** Qual extensão representa uma **imagem**?
A) .mp3 B) .avi C) .jpg D) .docx E) .exe
> ✔️ **C.** .jpg = imagem; .mp3 = áudio; .avi = vídeo; .exe = executável.

**INFO2-F05.** O formato **.mp4** é associado a:
A) texto. B) planilha. C) imagem estática. D) vídeo. E) áudio puro.
> ✔️ **D.** .mp4 = arquivo de **vídeo**.

**INFO2-F06.** A área de transferência (clipboard) armazena:
A) os arquivos excluídos. B) o conteúdo copiado ou recortado, para ser colado. C) as senhas salvas. D) o histórico de navegação. E) os downloads.
> ✔️ **B.** Clipboard = local temporário do que foi copiado/recortado.

**INFO2-F07.** O ícone de **nuvem** em aplicativos indica:
A) previsão do tempo. B) armazenamento ou sincronização online (cloud). C) conexão por cabo. D) backup local. E) antivírus ativo.
> ✔️ **B.** Ícone de nuvem = funcionalidade de **cloud**.

**INFO2-F08.** A unidade de medida de armazenamento **1 GB** equivale a:
A) 1.000 bytes. B) 1.000 MB. C) 1.024 MB. D) 1.000 KB. E) 1 TB.
> ✔️ **C.** 1 GB = **1.024 MB** (em binário; em decimal, fabricantes usam 1.000 MB).

**INFO2-F09.** A pasta **Downloads** no Windows geralmente armazena:
A) documentos criados pelo usuário. B) arquivos baixados da internet. C) arquivos do sistema. D) backups. E) drivers.
> ✔️ **B.** Downloads = arquivos recebidos da internet por padrão.

**INFO2-F10.** O que é um **driver**?
A) Um tipo de vírus. B) Software que permite ao sistema operacional se comunicar com um hardware. C) Um navegador de internet. D) Um tipo de planilha. E) Uma ferramenta de backup.
> ✔️ **B.** Driver = software de interface entre SO e hardware.

**INFO2-F11.** O botão **"Minimizar"** em uma janela do Windows:
A) fecha a janela. B) oculta a janela na barra de tarefas sem fechá-la. C) maximiza a janela. D) salva o arquivo. E) abre nova aba.
> ✔️ **B.** Minimizar = janela fica na barra de tarefas (não fechada).

**INFO2-F12.** O **processador (CPU)** é responsável por:
A) armazenar arquivos permanentemente. B) processar e executar as instruções dos programas. C) exibir imagens na tela. D) conectar à internet. E) alimentar o computador.
> ✔️ **B.** CPU = unidade central de processamento (executa instruções).

**INFO2-F13.** A função **"Salvar como"** difere de **"Salvar"** porque:
A) é mais lenta. B) permite escolher um novo nome, local ou formato para o arquivo. C) faz backup automático. D) envia o arquivo por e-mail. E) compacta o arquivo.
> ✔️ **B.** "Salvar como" = novo nome/local/formato; "Salvar" = sobrescreve o arquivo atual.

**INFO2-F14.** O protocolo **HTTP** significa:
A) High Transfer Technology Protocol. B) HyperText Transfer Protocol (protocolo de transferência de hipertexto). C) Hardware and Technology Transfer Program. D) Home Transfer Technology Port. E) HyperText Terminal Process.
> ✔️ **B.** HTTP = HyperText Transfer Protocol.

**INFO2-F15.** Uma URL típica de site seguro começa com:
A) ftp:// B) http:// C) https:// D) smtp:// E) ssh://
> ✔️ **C.** Site seguro (criptografado) = **https://**

**INFO2-F16.** O **teclado numérico** fica, geralmente, no lado:
A) esquerdo do teclado. B) central. C) direito do teclado. D) no mouse. E) na tela.
> ✔️ **C.** Teclado numérico = lado **direito** do teclado.

**INFO2-F17.** No Windows, pressionar **Windows + D** serve para:
A) abrir o disco D. B) exibir a área de trabalho (minimizando todas as janelas). C) abrir o gerenciador de dispositivos. D) desligar o computador. E) abrir o explorador de arquivos.
> ✔️ **B.** Windows + D = mostrar a **área de trabalho**.

**INFO2-F18.** O formato **PNG** de imagem suporta:
A) apenas cores sólidas. B) compressão com perda de qualidade. C) transparência (canal alfa). D) apenas imagens em preto e branco. E) vídeos embutidos.
> ✔️ **C.** PNG suporta **transparência** (diferente do JPEG, que não suporta).

**INFO2-F19.** O atalho **Ctrl+F** abre a ferramenta de:
A) formatação. B) busca/localizar no documento ou página. C) fechar. D) filtrar. E) favoritos.
> ✔️ **B.** Ctrl+F = **localizar** (Find).

**INFO2-F20.** A **impressora** é um periférico de:
A) entrada. B) entrada e saída. C) saída. D) armazenamento. E) processamento.
> ✔️ **C.** Impressora = periférico de **saída** (produz resultado físico).

**INFO2-F21.** O **scanner** é um periférico de:
A) saída. B) armazenamento. C) entrada. D) processamento. E) comunicação.
> ✔️ **C.** Scanner = periférico de **entrada** (digitaliza documentos físicos).

**INFO2-F22.** No Word, o atalho **Ctrl+Enter** insere:
A) um marcador de lista. B) uma quebra de página. C) uma tabela. D) um comentário. E) uma nota de rodapé.
> ✔️ **B.** Ctrl+Enter = **quebra de página** no Word.

**INFO2-M04.** No Word, o recurso **"Mala Direta"** (Correspondências) serve para:

**INFO2-M06.** A diferença entre **"animação"** e **"transição"** no PowerPoint é:

**INFO2-M07.** No contexto de redes, **WAN** refere-se a:

**INFO2-M08.** O conceito de **banda larga** refere-se a:

**INFO2-M09.** O **spyware** é um malware que:

**INFO2-M10.** O **adware** é um malware que:

**INFO2-M11.** Um ataque **DDoS (Distributed Denial of Service)** visa:
