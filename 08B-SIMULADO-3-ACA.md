# ✅ GABARITO COMENTADO

## Português
1. **B** — o texto mostra que o número é menor do que a história; dado ≠ esforço.
2. **B** — "menor" está em sentido conotativo: o número não captura toda a realidade.
3. **B** — a porteira evoca o contexto rural que precede o dado digital.
4. **A** — única forma graficamente correta ("necessita", "políticas").
5. **B** — "referiu-se **à** situação": "referir-se a" + artigo "a" (fem.) = crase obrigatória.
6. **B** — "a maioria" (singular) → verbo no singular ("chegou").
7. **B** — "assistir" (= presenciar) é transitivo indireto: "assistir **ao** treinamento".
8. **C** — "embora" indica concessão (contraste).
9. **C** — comparação implícita (memória = elefante) → **metáfora**.
10. **B** — tempo decorrido (passado) → "**Há** dez anos".
11. **C** — "Onde você vai agora?" usa "onde" para indicar movimento/destino, o que está incorreto; o correto seria "Aonde você vai?" ("aonde" = movimento; "onde" = lugar fixo).
12. **B** — "entregou **a ela**" → objeto indireto (preposição "a" + pronome).
13. **C** — "a fim de que" exprime **finalidade**.
14. **B** — sem vírgulas desnecessárias entre sujeito e verbo.
15. **C** — "ávido" e "desejoso" são sinônimos (os demais pares são antônimos).

18. **B** — diferenças: 3, 5, 7, 9 → +11: 26 + 11 = **37**.
19. **C** — 2000 × 1,12 = **R$ 2.240,00**.
20. **B** — só inglês = 30 − 10 = **20**.
21. **C** — 300 × 0,15 = **45**.
22. **C** — (4 + 8 + 12) ÷ 3 = 24 ÷ 3 = **8**.
23. **C** — todo A é B, e nenhum B é C → **nenhum A é C**.
24. **B** — bicondicional é verdadeira em VV e FF → **2** linhas.
25. **C** — 6 partes; cada parte = 50; maior (3) = **R$ 150,00**.
