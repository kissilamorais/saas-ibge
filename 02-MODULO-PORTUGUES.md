## AULA 1.1 — COMPREENSÃO E INTERPRETAÇÃO DE TEXTO

🎯 **Por que cai:** é o tópico nº 1 da IBFC. Espere **3 a 5 questões** só de interpretação. Domínio aqui = um quarto da nota de Português garantido.

### 📖 Resumo Teórico
Interpretar é diferente de compreender:
- **Compreensão** = o que o texto **diz** (informação explícita). Pergunta típica: "Segundo o texto…".
- **Interpretação** = o que o texto **permite concluir** (inferência). Pergunta típica: "Conclui-se que…", "Depreende-se que…".

**Regra de ouro da IBFC:** a resposta certa **está no texto** ou é uma conclusão **necessária** dele. Nunca traga conhecimento de fora ("eu acho", "na vida real é assim"). Se a alternativa exige informação que o texto não dá, está errada — por mais verdadeira que pareça.

**Tipos de questão e como atacar:**
1. **Ideia principal / tema** — o assunto central, geralmente no 1º ou último parágrafo. Não confunda **tema** (assunto amplo) com **ideia principal** (a tese sobre o tema).
2. **Informação explícita** — caça-palavra: localize no texto. Cuidado com sinônimos (a banca troca a palavra).
3. **Inferência** — conclusão obrigatória. Teste: "isso é *certo* a partir do texto ou só *possível*?". Só vale o que é certo.
4. **Sentido de palavra/expressão no contexto** — o significado **naquele** trecho, não no dicionário.
5. **Função de um trecho** — exemplificar, contrapor, concluir, justificar.

**Método dos 4 passos para cada questão:**
1. Leia o **enunciado primeiro** (saiba o que procurar).
2. Leia o texto **localizando** a resposta.
3. Elimine as alternativas com **erro de fato** (informação que contraria o texto).
4. Entre as que sobraram, escolha a que **não extrapola** (não vai além do texto).

**Vícios de alternativa errada (a IBFC usa muito):**
- **Extrapolação:** vai além do que o texto autoriza.
- **Generalização indevida:** troca "alguns" por "todos", "pode" por "deve".
- **Contradição:** afirma o oposto do texto.
- **Distorção:** mistura informações certas de jeito errado.

### 🧠 Mapa Mental
```
INTERPRETAÇÃO DE TEXTO
├── Compreensão → o que o texto DIZ (explícito)
├── Interpretação → o que se CONCLUI (inferência necessária)
├── Regra de ouro → resposta está no texto (nada de fora)
├── Tipos de pergunta
│   ├── ideia principal/tema
│   ├── informação explícita (cuidado: sinônimos)
│   ├── inferência (só o CERTO, não o possível)
│   └── sentido no contexto
└── Erros das alternativas
    ├── extrapola  ├── generaliza
    ├── contradiz  └── distorce
```

### 🃏 Flashcards
1. P: Diferença entre compreender e interpretar? **R:** Compreender = explícito (o que diz); interpretar = inferência (o que se conclui).
2. P: Posso usar conhecimento externo para responder? **R:** Não. A resposta vem do texto.
3. P: "Conclui-se que" pede o quê? **R:** Inferência (conclusão necessária).
4. P: Alternativa com "todos" quando o texto diz "alguns" é o quê? **R:** Generalização indevida (erro).
5. P: Sentido de uma palavra na prova é o do dicionário? **R:** Não, é o sentido no contexto.

### 🪄 Macetes & Pegadinhas IBFC
- **Macete da "alternativa verdadeira porém fora do texto":** a IBFC adora colocar uma alternativa **verdadeira no mundo real, mas não dita no texto**. É a pegadinha clássica. Verdade ≠ resposta. Só vale o que o texto sustenta.
- **Pegadinha do "exceto/incorreto":** sublinhe a palavra-negativa do enunciado. Muita gente perde a questão por ler "correto" quando pedia "incorreto".
- **Macete dos extremos:** alternativas com "sempre, nunca, todos, nenhum, exclusivamente" são **suspeitas** (textos costumam ser mais cuidadosos). Não é regra absoluta, mas é um forte indício.

### ✅ Questões Comentadas (estilo IBFC)
> *Texto-base (curto, para treino):* "O recenseamento agropecuário não se resume a contar propriedades. Ele revela quem produz o alimento do país, como produz e em que condições. Sem esses dados, políticas públicas para o campo seriam construídas no escuro."

**Q1.** De acordo com o texto, o recenseamento agropecuário:
A) limita-se a contar as propriedades rurais.
B) substitui as políticas públicas para o campo.
C) fornece dados que embasam políticas públicas para o campo. ✔️
D) é dispensável para o planejamento agrícola.
E) tem como único objetivo medir a produção de alimentos.

**Gabarito: C.** Comentário: o texto diz que sem os dados as políticas seriam feitas "no escuro" → logo, o censo **embasa** essas políticas (C). **A** contraria ("não se resume a contar"). **B** extrapola (o censo fornece dados, não substitui políticas). **D** contradiz. **E** generaliza indevidamente ("único objetivo").

**Q2.** Depreende-se do texto que:
A) o Brasil não possui políticas públicas para o campo.
B) dados de qualidade são condição para um bom planejamento público. ✔️
C) todos os produtores rurais participam do censo voluntariamente.
D) o alimento do país é produzido apenas por grandes propriedades.
E) o censo é mais importante que a própria produção agrícola.

**Gabarito: B.** Comentário: a metáfora "no escuro" leva à conclusão necessária de que **dados embasam bom planejamento** (B). As demais trazem informação **não contida** no texto (C, D) ou que **extrapolam/distorcem** (A, E).

## AULA 1.2 — SIGNIFICAÇÃO DAS PALAVRAS

🎯 **Por que cai:** 1 a 2 questões diretas. É **ponto fácil** — não dá para errar.

### 📖 Resumo Teórico
- **Sinônimos:** sentidos próximos (feliz / contente).
- **Antônimos:** sentidos opostos (claro / escuro).
- **Homônimos:** mesma forma (som e/ou grafia), sentidos diferentes:
  - *Homônimos homófonos* — mesmo som, grafia diferente (cela / sela; cessão / sessão / seção).
  - *Homônimos homógrafos* — mesma grafia, som diferente (gOsto/substantivo · gÓsto não — melhor: *colher* verbo / *colher* substantivo; *jogo* substantivo / *jogo* verbo).
  - *Homônimos perfeitos* — mesma grafia e som (manga da camisa / manga fruta).
- **Parônimos:** parecidos, mas diferentes (a grande pegadinha): *eminente* (notável) / *iminente* (prestes a ocorrer); *tráfego* (trânsito) / *tráfico* (comércio ilegal); *ratificar* (confirmar) / *retificar* (corrigir); *descrição* / *discrição*; *despensa* (lugar) / *dispensa* (ato de dispensar).
- **Polissemia:** uma palavra com vários sentidos relacionados (o *banco* da praça / *banco* financeiro).
- **Conotação x Denotação:** denotativo = sentido literal; conotativo = figurado ("ele tem um coração de ouro").

### 🧠 Mapa Mental
```
SIGNIFICAÇÃO
├── Sinônimo (igual) / Antônimo (oposto)
├── Homônimo (mesma forma)
│   ├── homófono (som ✓ grafia ✗) cela/sela
│   ├── homógrafo (grafia ✓ som ✗)
│   └── perfeito (som ✓ grafia ✓) manga/manga
├── Parônimo (parecido) eminente/iminente
├── Polissemia (1 palavra, vários sentidos ligados)
└── Denotação (literal) x Conotação (figurado)
```

### 🃏 Flashcards
1. P: *Iminente* significa? **R:** Prestes a acontecer.
2. P: *Eminente*? **R:** Notável, importante.
3. P: *Ratificar* x *retificar*? **R:** Ratificar = confirmar; retificar = corrigir.
4. P: Cela / sela são? **R:** Homônimos homófonos (som igual, grafia diferente).
5. P: *Tráfego* x *tráfico*? **R:** Tráfego = trânsito; tráfico = comércio ilegal.

### 🪄 Macetes & Pegadinhas IBFC
- **Macete dos parônimos:** monte uma lista pessoal dos pares clássicos e revise. A IBFC repete os mesmos (eminente/iminente, tráfego/tráfico, ratificar/retificar, descrição/discrição).
- **Macete "i de iminente = imediato/em cima":** *Iminente* tem "i" como em "imediato" → algo iminente está em cima de acontecer.
- **Pegadinha de sinônimo no contexto:** a banca pede o sinônimo **daquele uso**. "Manga" pode ser fruta ou parte da camisa — leia a frase.

### ✅ Questões Comentadas
**Q1.** "O coordenador alertou sobre o risco _______ de atraso no cronograma." A lacuna é preenchida por:
A) eminente B) iminente ✔️ C) emergente D) imanente E) preeminente
**Gabarito: B.** *Iminente* = prestes a ocorrer (combina com "risco de atraso"). *Eminente* (A) = notável, não cabe.

**Q2.** Assinale o par de **parônimos**:
A) feliz / contente B) claro / escuro C) descrição / discrição ✔️ D) manga / manga E) banco / banco
**Gabarito: C.** Descrição (relato) e discrição (reserva, recato) são parecidas mas diferentes → parônimos. A = sinônimos; B = antônimos; D = homônimos perfeitos; E = polissemia/homônimo.

## AULA 1.3 — ORTOGRAFIA E ACENTUAÇÃO

🎯 **Por que cai:** 1 a 2 questões. Tópico de **regra fixa** — estudou, acertou.

### 📖 Resumo Teórico
**Ortografia — casos que mais caem:**
- **Mas** (porém) x **mais** (adição/quantidade).
- **Mal** (advérbio, oposto de bem) x **mau** (adjetivo, oposto de bom). *Mal = adverbio (com L de "aLgo ruim")*.
- **A fim de** (finalidade) x **afim** (semelhante/parente).
- **Onde** (lugar fixo) x **aonde** (lugar com movimento — "para onde").
- **Por que / Por quê / Porque / Porquê:**
  - **Por que** (separado, sem acento): pergunta ou "pelo qual" → *Por que você faltou?*
  - **Por quê** (separado, com acento): fim de frase → *Você faltou por quê?*
  - **Porque** (junto): resposta/causa → *Faltei porque chovia.*
  - **Porquê** (junto, com acento): substantivo → *Não sei o porquê.*
- **Há** (verbo haver = existir/tempo passado) x **a** (preposição/tempo futuro): *Há dois anos* (atrás) x *Daqui a dois anos*.

**Acentuação gráfica (regras):**
- **Oxítonas** terminadas em **a(s), e(s), o(s), em, ens** → acentua: *sofá, café, cipó, também, parabéns*.
- **Paroxítonas** terminadas em **i(s), us, um, uns, l, n, r, x, ã(s), ão(s), ditongo** → acentua: *júri, vírus, álbum, fácil, hífen, caráter, tórax, ímã, órgão, série*.
- **Proparoxítonas:** **todas** são acentuadas: *médico, câmara, lâmpada*.
- **Hiato:** "i" e "u" tônicos sozinhos (ou com s) na sílaba, após vogal → *saída, baú, país*. (Não acentua se vier antes de "nh": *rainha*.)
- **Acordo Ortográfico (o que sumiu):** caiu o trema (exceto nomes próprios); não se acentuam mais **ditongos abertos "ei" e "oi"** em paroxítonas (*ideia, heroico, jiboia*); caiu o acento de "ôo"/"êem" (*voo, leem*).

### 🧠 Mapa Mental
```
ORTOGRAFIA & ACENTUAÇÃO
├── Pares clássicos
│   ├── mas(porém)/mais  • mal(adv)/mau(adj)
│   ├── a fim de(finalidade)/afim(semelhante)
│   ├── onde(parado)/aonde(movimento)
│   ├── PORQUE → por que | por quê | porque | porquê
│   └── há(haver/passado)/a(futuro)
└── Acentuação
    ├── oxítona: a,e,o,em,ens
    ├── paroxítona: i,us,um,l,n,r,x,ã,ditongo...
    ├── proparoxítona: TODAS
    ├── hiato: i/u tônicos sós (saída, baú)
    └── Acordo: sem trema, sem ei/oi de ideia/heroico, voo/leem
```

### 🃏 Flashcards
1. P: "Mal" ou "mau" para o oposto de "bem"? **R:** Mal (advérbio).
2. P: *Por que* no fim da frase leva acento? **R:** Sim → "por quê".
3. P: Proparoxítona acentua-se quando? **R:** Sempre (todas).
4. P: "Ideia" leva acento? **R:** Não (caiu pelo Acordo).
5. P: "Há dois anos" indica passado ou futuro? **R:** Passado (verbo haver). Futuro = "daqui a".

### 🪄 Macetes & Pegadinhas IBFC
- **Macete "maL = aLgo ruim (advérbio); maU = bom↔mau (adjetivo)".** Troque por "bem/bom": se cabe "bem" → mal; se cabe "bom" → mau.
- **Macete do "porquê":** se dá para colocar **artigo** na frente ("o porquê"), é junto e com acento. Se está **no fim da frase**, leva acento ("por quê").
- **Pegadinha do "a/há tempo":** "Faz/Há dois anos" (passado) x "Daqui a dois anos" (futuro). A IBFC mistura na mesma frase.

### ✅ Questões Comentadas
**Q1.** Assinale a frase **correta**:
A) Ele se comportou mau na reunião.
B) Faltei mais não avisei.
C) Não sei aonde guardei o documento.
D) Faltei porque estava doente. ✔️
E) Você chegou atrasado por que?
**Gabarito: D.** "Porque" (junto) = causa/resposta. Erros: A) "mal" (advérbio); B) "mas" (porém); C) "onde" (lugar sem movimento); E) "por quê" (fim de frase, com acento).

**Q2.** Todas as palavras estão corretamente acentuadas, EXCETO:
A) médico B) sofá C) ideia D) idéia ✔️ E) saída
**Gabarito: D.** Pelo Acordo Ortográfico, "ideia" **não** leva acento (C está certa; D, com acento, está errada).

## AULA 1.4 — PONTUAÇÃO

🎯 **Por que cai:** 1 a 2 questões, frequentemente ligadas à vírgula e à reescrita.

### 📖 Resumo Teórico
**A VÍRGULA — onde USAR:**
- Separar **itens de enumeração**: *Comprei caneta, papel e cola.*
- Isolar **aposto** e **vocativo**: *João, o supervisor, chegou.* / *Atenção, candidatos!*
- Isolar **adjuntos adverbiais deslocados** (sobretudo no início): *Ontem, fui à prova.*
- Separar **orações coordenadas** (exceto as ligadas por "e" em regra): *Estudei, mas não passei.*
- Isolar **orações intercaladas/explicativas**.

**A VÍRGULA — onde É PROIBIDO usar:** **nunca** separe:
- Sujeito do verbo: ❌ *Os candidatos, estudaram.*
- Verbo do seu complemento (objeto): ❌ *Ele comprou, um livro.*
- Nome do seu adjunto/complemento nominal.

**Outros sinais:**
- **Ponto e vírgula (;):** separa orações longas já com vírgulas; itens de lista/enumeração de lei.
- **Dois-pontos (:):** anuncia enumeração, fala (citação) ou explicação.
- **Travessão:** marca fala em diálogo ou isola termo (como a vírgula dupla).
- **Aspas:** citação, ironia, gíria, estrangeirismo.

### 🧠 Mapa Mental
```
PONTUAÇÃO
├── VÍRGULA usa-se p/
│   ├── enumeração
│   ├── aposto/vocativo (isolar)
│   ├── adjunto adverbial deslocado
│   └── orações coordenadas (mas, porém, contudo)
├── VÍRGULA proibida (NUNCA separar)
│   ├── sujeito × verbo
│   └── verbo × objeto
├── ; → orações longas / itens de lei
├── : → anuncia (fala, lista, explicação)
└── travessão / aspas
```

### 🃏 Flashcards
1. P: Posso separar sujeito do verbo por vírgula? **R:** Nunca.
2. P: Vírgula antes de "mas"? **R:** Sim (oração coordenada adversativa).
3. P: O que os dois-pontos anunciam? **R:** Enumeração, fala ou explicação.
4. P: Adjunto adverbial no início da frase pede vírgula? **R:** Em geral sim ("Ontem, …").
5. P: Para que serve o travessão? **R:** Marcar fala ou isolar termo.

### 🪄 Macetes & Pegadinhas IBFC
- **Macete da "tríade proibida":** sujeito–verbo–objeto é uma corrente que **não se quebra** com vírgula. Se a vírgula está no meio dessa corrente, está errada.
- **Pegadinha do "e" com vírgula:** normalmente não há vírgula antes de "e", **mas há** quando o "e" liga orações de sujeitos diferentes ou tem valor adversativo ("Estudou muito, e mesmo assim não passou").
- **Pegadinha da reescrita:** ao deslocar o adjunto adverbial para o início, **acrescenta-se vírgula**; ao voltar ao fim, **tira-se**.

### ✅ Questões Comentadas
**Q1.** Assinale a opção com pontuação **correta**:
A) Os agentes do censo, percorreram a zona rural.
B) Ontem, a equipe concluiu a coleta de dados. ✔️
C) Ele entregou ao supervisor, o relatório.
D) Comprei, caneta, papel e cola.
E) João o coordenador chegou cedo.
**Gabarito: B.** Adjunto adverbial deslocado ("Ontem") corretamente isolado. Erros: A) separa sujeito de verbo; C) separa verbo de objeto; D) vírgula indevida após "comprei"; E) falta isolar o aposto ("João, o coordenador, chegou").

## AULA 1.5 — CLASSES DE PALAVRAS

🎯 **Por que cai:** tópico **pesado** — 2 a 4 questões. Base para concordância, regência e sintaxe.

### 📖 Resumo Teórico
As **10 classes** (6 variáveis, 4 invariáveis):

**Variáveis:**
1. **Substantivo** — nomeia (casa, censo, alegria). Próprio/comum, concreto/abstrato, coletivo.
2. **Artigo** — define (o, a) ou indefine (um, uma) o substantivo.
3. **Adjetivo** — qualifica o substantivo (agente *competente*).
4. **Numeral** — quantidade/ordem (dois, segundo, dobro).
5. **Pronome** — substitui/acompanha o nome. Tipos: pessoais (eu, me, ele), possessivos (meu), demonstrativos (este, esse, aquele), indefinidos (algum, todo), relativos (que, o qual, cujo), interrogativos (quem, qual).
6. **Verbo** — ação, estado ou fenômeno (estudar, ser, chover).

**Invariáveis:**
7. **Advérbio** — modifica verbo/adjetivo/outro advérbio (muito, ontem, aqui, não). Circunstância: tempo, lugar, modo, intensidade, negação, afirmação.
8. **Preposição** — liga termos (de, para, com, em, a, por). Essenciais x acidentais.
9. **Conjunção** — liga orações. Coordenativas (e, mas, ou, porém, portanto) e subordinativas (que, porque, se, embora, quando).
10. **Interjeição** — emoção (Oba! Ufa! Cuidado!).

**Cuidado — palavras que mudam de classe conforme o contexto:**
- *que*: pronome relativo, conjunção, partícula expletiva…
- *a*: artigo, preposição, pronome.
- Diferencie **adjetivo** de **advérbio**: adjetivo se refere a substantivo (varia em gênero/número); advérbio se refere a verbo (invariável).

### 🧠 Mapa Mental
```
CLASSES DE PALAVRAS (10)
├── VARIÁVEIS (6)
│   ├── substantivo (nomeia)
│   ├── artigo (define/indefine)
│   ├── adjetivo (qualifica)
│   ├── numeral (quantidade/ordem)
│   ├── pronome (subst./acompanha)
│   │   └── pessoal,possessivo,demonstr.,indef.,relativo,interrog.
│   └── verbo (ação/estado/fenômeno)
└── INVARIÁVEIS (4)
    ├── advérbio (modifica verbo/adj/adv)
    ├── preposição (liga termos)
    ├── conjunção (liga orações: coord. × subord.)
    └── interjeição (emoção)
```

### 🃏 Flashcards
1. P: Quantas classes variáveis? **R:** 6 (subst., artigo, adjetivo, numeral, pronome, verbo).
2. P: Advérbio varia em gênero/número? **R:** Não (invariável).
3. P: "Cujo" é que tipo de pronome? **R:** Relativo (indica posse).
4. P: Conjunção liga o quê? **R:** Orações (ou termos de mesma função).
5. P: Diferença prática adjetivo x advérbio? **R:** Adjetivo → liga-se a substantivo e varia; advérbio → liga-se a verbo e é invariável.

### 🪄 Macetes & Pegadinhas IBFC
- **Macete das invariáveis: "PIAC"** → **P**reposição, **I**nterjeição, **A**dvérbio, **C**onjunção. O resto varia.
- **Pegadinha do "que":** a IBFC pede a classe do "que" na frase. Teste: se equivale a "o qual" → pronome relativo; se introduz oração subordinada → conjunção; se pode ser retirado sem mudar o sentido essencial → expletivo/realce.
- **Pegadinha adjetivo↔advérbio:** "Falou *claro*" (advérbio, = claramente) x "Texto *claro*" (adjetivo). Se modifica verbo e está invariável → advérbio.
- **Macete do pronome relativo "cujo" = posse:** "cujo" sempre indica posse e **não vem com artigo depois** (❌ "cujo o"). "O autor *cujas* obras…".

### ✅ Questões Comentadas
**Q1.** Na frase "Os agentes trabalharam **muito** durante o censo", a palavra destacada é:
A) adjetivo B) advérbio ✔️ C) numeral D) pronome E) preposição
**Gabarito: B.** "Muito" modifica o verbo "trabalharam" (intensidade) → advérbio. Seria adjetivo em "muito*s* agentes" (variando = pronome indefinido/numeral, não advérbio).

**Q2.** "O servidor **cujo** relatório foi elogiado recebeu um prêmio." O termo destacado é pronome:
A) demonstrativo B) pessoal C) relativo ✔️ D) indefinido E) possessivo
**Gabarito: C.** "Cujo" é pronome relativo (liga orações e indica posse: o relatório *do* servidor).

## AULA 1.6 — EMPREGO DOS PRONOMES

🎯 **Por que cai:** 1 a 2 questões; também aparece dentro de interpretação (referência/coesão).

### 📖 Resumo Teórico
- **Demonstrativos e a "regra do espaço/tempo":**
  - **este/esta/isto** → perto de quem fala / tempo presente ou futuro / o que vai ser dito.
  - **esse/essa/isso** → perto de quem ouve / passado próximo / o que já foi dito.
  - **aquele/aquela/aquilo** → distante de ambos / passado remoto.
  - Em texto: *este* retoma o que vem **depois** (catáfora) ou o **último** citado; *aquele*, o **primeiro/mais distante**.
- **Pronomes de tratamento** (concordam em **3ª pessoa**): Vossa Senhoria *fez* (não "fizestes"). V. Sª, V. Exª.
- **Colocação pronominal** (próclise, ênclise, mesóclise):
  - **Próclise** (pronome antes do verbo) — atraída por: palavra negativa (não, nunca), advérbio, pronome relativo/indefinido/interrogativo, conjunção subordinativa. *Não **me** disseram.*
  - **Ênclise** (depois) — início de oração e imperativo afirmativo. *Disseram-**me** a verdade.*
  - **Mesóclise** (no meio) — futuro do presente/pretérito sem fator de próclise. *Dir-**se**-á a verdade.*
- **Possessivos:** concordam com a coisa possuída (meu livro, minha caneta).

### 🧠 Mapa Mental
```
PRONOMES
├── Demonstrativos
│   ├── este/isto → perto de quem fala / futuro / vai dizer
│   ├── esse/isso → perto de quem ouve / já dito
│   └── aquele/aquilo → distante / passado remoto
├── Tratamento → concorda em 3ª pessoa (V.Sª fez)
└── Colocação
    ├── PRÓCLISE (antes): palavra atrativa (não, que, advérbio)
    ├── ÊNCLISE (depois): início de frase / imperativo +
    └── MESÓCLISE (meio): futuro sem atração
```

### 🃏 Flashcards
1. P: Qual demonstrativo retoma algo já dito? **R:** Esse/essa/isso.
2. P: Pronome de tratamento exige verbo em que pessoa? **R:** 3ª pessoa.
3. P: Palavra negativa antes do verbo pede próclise ou ênclise? **R:** Próclise ("não me viu").
4. P: Pode começar frase com pronome oblíquo átono? **R:** Não → usa-se ênclise.
5. P: "Meu/minha" concorda com quem? **R:** Com a coisa possuída.

### 🪄 Macetes & Pegadinhas IBFC
- **Macete "palavra que atrai, próclise vai":** havendo palavra atrativa (não, nunca, que, quem, advérbio, conjunção subordinada), o pronome vem **antes**. *"Quem **te** avisou?"*
- **Pegadinha do início de frase:** ❌ "*Me* empresta a caneta" (coloquial). Na norma culta → "Empresta-**me**". A IBFC cobra a norma.
- **Pegadinha do tratamento:** ❌ "Vossa Excelência sois bondoso" → ✔️ "Vossa Excelência **é** bondoso" (3ª pessoa).

### ✅ Questões Comentadas
**Q1.** Assinale a colocação pronominal adequada à norma culta:
A) Me disseram que a prova mudou.
B) Não disseram-me a verdade.
C) Disseram-me que a prova mudou. ✔️
D) Que disseram-me?
E) Nunca contaram-me isso.
**Gabarito: C.** Ênclise correta (início de oração, sem fator de próclise). A) não inicia frase com oblíquo; B), D) e E) têm palavra atrativa ("não", "que", "nunca") → exigiriam próclise: "não **me** disseram", "que **me** disseram", "nunca **me** contaram".

## AULA 1.7 — VERBOS: EMPREGO E VOZES

🎯 **Por que cai:** 1 a 3 questões (tempos/modos e vozes). Também é base da concordância verbal.

### 📖 Resumo Teórico
**Conjugação — o essencial:**
- **Modos:** Indicativo (certeza), Subjuntivo (hipótese/desejo: "que eu *estude*"), Imperativo (ordem).
- **Tempos do indicativo:** presente, pretérito (perfeito/imperfeito/mais-que-perfeito), futuro (do presente/do pretérito).
- **Verbos regulares** (seguem o modelo: cantar, vender, partir), **irregulares** (mudam o radical: fazer→fiz, trazer→trouxe) e **anômalos** (ser, ir — fogem totalmente do padrão).
- **Pegadinhas de conjugação clássicas:** *ver* (que ele veja, se ele vir) x *vir* (que ele venha, se ele vier); *pôr* (puseram, eu punha); *intervir* (interveio, se intervier).

**Vozes verbais:**
- **Voz ativa:** sujeito pratica a ação. *O agente coletou os dados.*
- **Voz passiva:** sujeito sofre a ação.
  - *Analítica:* ser + particípio → *Os dados **foram coletados** pelo agente.*
  - *Sintética/pronominal:* verbo + **se** (partícula apassivadora) → *Coletaram-se os dados.* / *Alugam-se casas.*
- **Voz reflexiva:** sujeito pratica e sofre. *O candidato **se** preparou.*
- **Conversão ativa↔passiva:** o objeto direto da ativa vira sujeito da passiva.

### 🧠 Mapa Mental
```
VERBOS
├── Modos: Indicativo(certeza) • Subjuntivo(hipótese) • Imperativo(ordem)
├── Classes: regular • irregular(fiz, trouxe) • anômalo(ser, ir)
├── Pegadinhas: ver×vir | pôr(puseram) | intervir
└── VOZES
    ├── ativa (sujeito faz)
    ├── passiva analítica (ser+particípio) "foram coletados"
    ├── passiva sintética (verbo+SE) "coletaram-se os dados"
    └── reflexiva (faz e sofre) "se preparou"
```

### 🃏 Flashcards
1. P: "Se ele _____ o documento" (ver) → **R:** vir ("se ele vir").
2. P: "Quando ele _____" (vir/chegar) → **R:** vier.
3. P: Passiva analítica se forma como? **R:** Verbo SER + particípio.
4. P: "Vendem-se casas" está em que voz? **R:** Passiva sintética (pronominal).
5. P: Modo da hipótese/desejo? **R:** Subjuntivo.

### 🪄 Macetes & Pegadinhas IBFC
- **Macete "passiva sintética = some o sujeito agente":** "Alugam-se casas" → quem aluga não aparece; "casas" é o **sujeito paciente** (por isso o verbo vai pro plural: aluga**m**-se casa**s**). É a pegadinha de concordância nº 1 da banca.
- **Pegadinha ver × vir:** futuro do subjuntivo de **ver** = "vir" (se eu vir); de **vir** = "vier" (se eu vier). Memorize: *VER vira "vir"; VIR vira "vier"*.
- **Pegadinha do imperativo:** o imperativo afirmativo de "tu" vem do presente do indicativo **sem o -s** (tu cantas → canta tu).

### ✅ Questões Comentadas
**Q1.** "______ muitas casas no novo loteamento." Assinale a forma correta na voz passiva sintética:
A) Vende-se B) Vendem-se ✔️ C) É vendido D) Vende E) Vendeu-se
**Gabarito: B.** "Casas" é sujeito paciente, plural → o verbo concorda no plural: "Vende**m**-se casas". É o erro clássico que a IBFC cobra.

**Q2.** Transpondo "O recenseador entregou os formulários" para a voz passiva analítica:
A) Os formulários entregaram o recenseador.
B) Entregaram-se os formulários.
C) Os formulários foram entregues pelo recenseador. ✔️
D) O recenseador foi entregue pelos formulários.
E) Os formulários eram entregues.
**Gabarito: C.** Objeto direto ("os formulários") vira sujeito; verbo SER + particípio ("foram entregues"); agente da passiva ("pelo recenseador"). B é passiva sintética; as demais distorcem o sentido.

## AULA 1.8 — SINTAXE DA ORAÇÃO

🎯 **Por que cai:** 2 a 3 questões (sujeito, objeto, predicado). Fundamento da concordância e regência.

### 📖 Resumo Teórico
**Termos essenciais:**
- **Sujeito:** de quem se declara algo. Tipos: simples, composto, oculto/desinencial, indeterminado (verbo na 3ª pl. ou + SE), inexistente (orações sem sujeito: "Há vagas", "Choveu").
- **Predicado:** o que se diz do sujeito. Verbal (núcleo = verbo de ação), nominal (núcleo = predicativo, com verbo de ligação), verbo-nominal (dois núcleos).

**Termos integrantes (completam):**
- **Objeto direto** (sem preposição obrigatória): *Comprei **o livro**.*
- **Objeto indireto** (com preposição): *Gosto **de música**.*
- **Complemento nominal** (completa nome, com preposição): *Tenho necessidade **de ajuda**.*
- **Agente da passiva:** *...foi feito **pelo aluno**.*

**Termos acessórios:**
- **Adjunto adnominal** (liga-se a substantivo: artigo, adjetivo, pronome): *o **meu** novo carro*.
- **Adjunto adverbial** (circunstância: tempo, lugar, modo…): *Cheguei **cedo**.*
- **Aposto** (explica/identifica): *Brasília, **capital do país**...*
- **Vocativo** (chama): ***Candidatos**, atenção!*

**Verbo de ligação x verbo nocional:** ser, estar, ficar, parecer, permanecer (ligam sujeito a predicativo) → predicado nominal.

### 🧠 Mapa Mental
```
SINTAXE DA ORAÇÃO
├── ESSENCIAIS
│   ├── sujeito (simples/composto/oculto/indeterminado/inexistente)
│   └── predicado (verbal / nominal / verbo-nominal)
├── INTEGRANTES (completam)
│   ├── obj. direto (sem prep.)  • obj. indireto (com prep.)
│   ├── complemento nominal (completa NOME)
│   └── agente da passiva (pelo/pela)
└── ACESSÓRIOS
    ├── adj. adnominal (liga a substantivo)
    ├── adj. adverbial (circunstância)
    ├── aposto (explica) • vocativo (chama)
```

### 🃏 Flashcards
1. P: Objeto direto tem preposição obrigatória? **R:** Não.
2. P: Verbo de ligação forma que predicado? **R:** Nominal.
3. P: "Há vagas" — qual o tipo de sujeito? **R:** Inexistente (oração sem sujeito).
4. P: Diferença adjunto adnominal x complemento nominal? **R:** Adnominal liga-se a substantivo concreto e pode ser adjetivo/pronome; complemento nominal completa o sentido de nome abstrato e é sempre com preposição (tem valor passivo).
5. P: Vocativo pertence à oração? **R:** Não — é termo isolado, à parte da estrutura.

### 🪄 Macetes & Pegadinhas IBFC
- **Macete sujeito indeterminado:** verbo na **3ª pessoa do plural** sem sujeito claro ("Roubaram minha bolsa") **ou** verbo + **SE** com verbo intransitivo/de ligação ("Precisa-se de funcionários" → "Vive-se bem aqui").
- **Pegadinha "objeto direto preposicionado":** alguns objetos diretos vêm com preposição por ênfase/clareza ("Amar **a** Deus"). Continua sendo objeto **direto**.
- **Pegadinha adnominal × complemento nominal:** "amor **de mãe**" (mãe ama = agente → adnominal) x "amor **à pátria**" (alguém ama a pátria = paciente → complemento nominal). Quem pratica = adnominal; quem recebe = complemento nominal.

### ✅ Questões Comentadas
**Q1.** Em "Os agentes entregaram os relatórios ao supervisor", os termos sublinhados são, respectivamente:
A) sujeito, objeto direto, objeto indireto ✔️
B) sujeito, objeto indireto, objeto direto
C) objeto direto, sujeito, adjunto
D) sujeito, complemento nominal, aposto
E) predicativo, objeto direto, vocativo
**Gabarito: A.** "Os agentes" = sujeito; "os relatórios" = objeto direto (sem preposição); "ao supervisor" = objeto indireto (com preposição "a").

**Q2.** "Choveu muito ontem na zona rural." O sujeito desta oração é:
A) simples B) composto C) oculto D) indeterminado E) inexistente ✔️
**Gabarito: E.** Verbo que indica fenômeno da natureza ("chover") é impessoal → **oração sem sujeito** (sujeito inexistente).

## AULA 1.9 — CONCORDÂNCIA NOMINAL E VERBAL

🎯 **Por que cai:** tópico **campeão de eliminação** — 2 a 4 questões. Estude com prioridade máxima.

### 📖 Resumo Teórico
**CONCORDÂNCIA VERBAL (verbo concorda com o sujeito):**
- Sujeito composto **antes** do verbo → plural: *João e Maria **chegaram**.*
- Sujeito composto **depois** do verbo → pode concordar com o mais próximo ou ir ao plural: *Chegou/Chegaram João e Maria.*
- **"Haver" no sentido de existir = impessoal** → fica no **singular**: *Havia muitas vagas.* (❌ "Haviam"). Mas **"existir" concorda**: *Existiam muitas vagas.*
- **"Fazer" indicando tempo = impessoal** → singular: *Faz dois anos.*
- **Verbo + SE (partícula apassivadora)** concorda com o sujeito paciente: *Vendem-se casas.* Já **SE índice de indeterminação** (com VI/VL/VTI) → verbo no **singular**: *Precisa-se de funcionários.*
- **% e expressões de quantidade:** o verbo concorda com o numeral ou com o nome seguinte: *30% dos candidatos faltaram / 1% faltou.*
- **Sujeito coletivo no singular** → verbo no singular: *A multidão gritou.*

**CONCORDÂNCIA NOMINAL (adjetivo/artigo concordam com o substantivo):**
- Adjetivo após vários substantivos → concorda com o mais próximo ou vai ao plural masculino: *Comprei livro e revista **nova/novos**.*
- Adjetivo antes de vários substantivos → concorda com o mais próximo: ***Velho** livro e revista.*
- **Palavras invariáveis** (cuidado!): *menos* (nunca "menas"), *alerta*, *pseudo*.
- **"Anexo, incluso, mesmo, próprio, obrigado"** variam (concordam): *Seguem **anexos** os documentos.* / *Elas **mesmas** fizeram.* / Mulher diz "**obrigada**".
- **"É proibido / é necessário / é bom"** → invariável sem artigo; variável com artigo: *É proibido entrada* x *É proibida **a** entrada.*
- **"Meio"** = "um pouco" (advérbio, invariável): *Ela está **meio** cansada* (não "meia").

### 🧠 Mapa Mental
```
CONCORDÂNCIA
├── VERBAL (verbo ↔ sujeito)
│   ├── sujeito composto antes → plural
│   ├── HAVER (existir) → SINGULAR (havia vagas)
│   ├── EXISTIR → concorda (existiam vagas)
│   ├── FAZER (tempo) → singular (faz 2 anos)
│   ├── verbo+SE apassivador → concorda (vendem-se casas)
│   └── SE indeterminação (VI/VTI/VL) → singular (precisa-se de)
└── NOMINAL (adjetivo ↔ substantivo)
    ├── adj. depois → + próximo ou plural masc.
    ├── invariáveis: menos, alerta, pseudo
    ├── anexo/incluso/mesmo/obrigado → VARIAM
    ├── proibido/necessário → varia SÓ com artigo
    └── meio = "um pouco" → invariável (meio cansada)
```

### 🃏 Flashcards
1. P: "Havia/Haviam muitas pessoas"? **R:** Havia (haver = existir é impessoal, singular).
2. P: "Existia/Existiam muitas pessoas"? **R:** Existiam (existir concorda).
3. P: "É proibido a entrada" — certo? **R:** Não → "É proibida a entrada" (com artigo, concorda).
4. P: "Menas pessoas" existe? **R:** Não. "Menos" é invariável.
5. P: Mulher agradece dizendo? **R:** "Obrigada" (concorda).
6. P: "Ela está meia/meio cansada"? **R:** Meio (= um pouco, advérbio invariável).

### 🪄 Macetes & Pegadinhas IBFC
- **Macete do HAVER:** "haver" no sentido de existir é **solteiro** — fica sozinho no singular, **nunca** vai ao plural. Troque por "existir" para conferir o sentido, mas **mantenha** "havia" no singular. É a pegadinha nº 1 da banca.
- **Macete "OBRIGADA/OBRIGADO concorda com quem fala":** homem fala "obrigado", mulher fala "obrigada".
- **Pegadinha do "fazer tempo":** "Faz**em** cinco anos" está errado → "Faz cinco anos" (impessoal, singular).
- **Pegadinha do "anexo":** "Segue anexo a nota" → ❌. "Segue anexa a nota" (concorda com "nota"). "Seguem anexos os documentos".
- **Pegadinha "é bom / é necessário":** sem artigo → invariável ("Água é bom"); com determinante → concorda ("A água é boa").

### ✅ Questões Comentadas
**Q1.** Assinale a frase **correta** quanto à concordância:
A) Haviam muitos candidatos na sala.
B) Existia poucas vagas na cidade.
C) Faz cinco anos que o censo não é realizado. ✔️
D) Segue anexo as instruções.
E) É proibido a permanência de pessoas estranhas.
**Gabarito: C.** "Fazer" indicando tempo é impessoal → singular ("faz cinco anos"). Erros: A) "Havia" (haver = existir, singular); B) "Existiam" (existir concorda); D) "anexas as instruções"; E) "É proibida a permanência" (com artigo, concorda).

**Q2.** "_____ muitas dúvidas durante a aplicação da prova." Complete corretamente:
A) Houveram B) Houve ✔️ C) Existiu D) Tiveram E) Haviam
**Gabarito: B.** "Haver" no sentido de existir é impessoal e fica no **singular**, mesmo no passado: "Houve muitas dúvidas". "Houveram" não existe nesse sentido.

## AULA 1.10 — REGÊNCIA E CRASE

🎯 **Por que cai:** 1 a 3 questões; a **crase** é a estrela das pegadinhas da IBFC.

### 📖 Resumo Teórico
**REGÊNCIA = a preposição que o verbo/nome exige.**
- **Verbos clássicos cobrados:**
  - *Assistir* (= ver) → **a**: *Assisti **ao** filme.* (Assistir = ajudar/morar tem outras regências.)
  - *Aspirar* (= desejar) → **a**: *Aspiro **a** um cargo.* (= cheirar é direto.)
  - *Visar* (= almejar) → **a**: *Viso **a** uma vaga.* (= mirar/dar visto é direto.)
  - *Obedecer / desobedecer* → **a**: *Obedeço **às** regras.*
  - *Implicar* (= acarretar) → **direto**: *O atraso implica perdas.* (Sem "em".)
  - *Namorar* → **direto**: *Namora João* (não "com").
  - *Preferir* → "preferir A a B": *Prefiro café a chá* (não "do que").

**CRASE = fusão da preposição "a" + artigo "a(s)" → à / às.** Ocorre quando há, ao mesmo tempo, uma palavra que **exige a preposição "a"** e uma palavra **que admite o artigo "a"**.
- **HÁ crase:** *Vou **à** escola* (ir a + a escola); *Refiro-me **à** professora*; *das 8 **às** 18h*.
- **NÃO há crase:**
  - Antes de **palavra masculina**: *Vou **a** pé.* (truque: troque por palavra feminina → "à").
  - Antes de **verbo**: *Começou **a** estudar.*
  - Antes de **pronomes** (em regra): *Falei **a** ela*, *a esta, a você*.
  - Antes de palavra no **plural** quando o "a" está no singular: *Refiro-me **a** pessoas.*
- **Casos especiais:**
  - *À distância* (sem complemento) x *a distância de 10m* (com complemento) — atenção ao uso.
  - *Às vezes* (locução) → **sempre com crase**.
  - *À moda de*: *bife **à** (moda) francesa* → crase mesmo antes de masculino, pois subentende "à moda de".

### 🧠 Mapa Mental
```
REGÊNCIA & CRASE
├── REGÊNCIA (preposição exigida)
│   ├── assistir(ver)/aspirar(desejar)/visar(almejar) → A
│   ├── obedecer → A | implicar(acarretar) → direto
│   └── preferir A a B (nunca "do que")
└── CRASE = a (prep.) + a (artigo) = À
    ├── HÁ: vou À escola | das 8 ÀS 18h | ÀS vezes
    └── NÃO há:
        ├── antes de masculino (a pé)
        ├── antes de verbo (a estudar)
        ├── antes de pronome (a ela)
        └── TRUQUE: troque por palavra masculina
            → vira "ao"? então tem crase ("à")
```

### 🃏 Flashcards
1. P: "Assisti ___ filme" → **R:** ao (assistir = ver pede "a").
2. P: Tem crase antes de verbo? **R:** Não ("começou a estudar").
3. P: "Às vezes" tem crase? **R:** Sempre.
4. P: Truque da crase? **R:** Troque a palavra feminina por uma masculina: se virar "ao", há crase.
5. P: "Prefiro café ___ chá" → **R:** "a" (preferir A a B; nunca "do que").

### 🪄 Macetes & Pegadinhas IBFC
- **Macete do truque "AO":** substitua a palavra feminina por uma masculina equivalente. "Vou à feira" → "Vou ao mercado" (deu "ao") → **tem crase**. "Vou a pé" → "Vou a/ao pé"? Não combina → sem crase.
- **Pegadinha "a partir de":** **nunca** tem crase ("a partir de hoje" — antes de verbo/expressão sem artigo).
- **Pegadinha do masculino:** "Andou a cavalo", "Fui a pé", "Vendo a prazo" → sem crase (palavra masculina).
- **Pegadinha "à/a ela":** antes de pronome pessoal não há crase: "Entreguei a ela".

### ✅ Questões Comentadas
**Q1.** Assinale a frase em que o uso da crase está **correto**:
A) Vou à pé para a prova.
B) Assisti à o filme ontem.
C) O expediente é das 8h às 17h. ✔️
D) Começou à chover.
E) Entreguei o documento à ela.
**Gabarito: C.** "Das 8 às 17h" — crase em expressão de horário determinado. Erros: A) "a pé" (masculino, sem crase); B) "ao filme" (não existe "à o"); D) antes de verbo não há crase ("a chover"); E) antes de pronome ("a ela").

**Q2.** "Os recenseadores devem obedecer ___ normas do IBGE e visar ___ qualidade dos dados." Complete:
A) as / a B) às / à ✔️ C) as / à D) às / a E) à / às
**Gabarito: B.** "Obedecer **a**" + "as normas" = às normas; "visar (almejar) **a**" + "a qualidade" = à qualidade. Ambos os verbos pedem preposição "a", e os substantivos femininos admitem artigo → crase nos dois.

## AULA 1.11 — COESÃO E COERÊNCIA

🎯 **Por que cai:** 1 a 3 questões; conecta-se diretamente com interpretação e reescrita.

### 📖 Resumo Teórico
- **Coesão** = ligação entre as partes do texto (a "costura"). Mecanismos:
  - **Referenciação:** pronomes e expressões retomam termos (anáfora: retoma o que já foi dito; catáfora: antecipa). *"O agente chegou. **Ele** trouxe os dados."*
  - **Substituição / repetição:** trocar por sinônimo ou pronome para evitar repetir.
  - **Conectores (conjunções/preposições):** ligam ideias indicando relação lógica.
- **Coerência** = sentido lógico global; o texto não se contradiz e faz sentido como um todo.
- **Conectores e suas relações lógicas (MUITO cobrado):**
  - **Adição:** e, além disso, também, ademais.
  - **Oposição/contraste:** mas, porém, contudo, todavia, entretanto, no entanto.
  - **Causa:** porque, pois, já que, visto que, uma vez que.
  - **Consequência/conclusão:** portanto, logo, assim, por isso, então.
  - **Condição:** se, caso, contanto que, desde que.
  - **Finalidade:** para que, a fim de que.
  - **Tempo:** quando, enquanto, assim que, antes que.
  - **Concessão (quebra de expectativa):** embora, ainda que, mesmo que, apesar de.

### 🧠 Mapa Mental
```
COESÃO & COERÊNCIA
├── COESÃO (costura)
│   ├── referenciação (anáfora ↩ / catáfora ↪)
│   ├── substituição / repetição
│   └── conectores (relação lógica)
├── COERÊNCIA (sentido global, sem contradição)
└── CONECTORES por sentido
    ├── adição: e, além disso
    ├── oposição: mas, porém, contudo, todavia
    ├── causa: porque, já que, visto que
    ├── conclusão: portanto, logo, por isso
    ├── condição: se, caso, desde que
    ├── finalidade: para que, a fim de que
    └── concessão: embora, ainda que, apesar de
```

### 🃏 Flashcards
1. P: "Embora" indica que relação? **R:** Concessão (quebra de expectativa).
2. P: "Portanto" indica? **R:** Conclusão/consequência.
3. P: Anáfora x catáfora? **R:** Anáfora retoma (para trás); catáfora antecipa (para frente).
4. P: "Visto que" indica? **R:** Causa.
5. P: Diferença coesão x coerência? **R:** Coesão = ligação formal entre partes; coerência = sentido lógico do todo.

### 🪄 Macetes & Pegadinhas IBFC
- **Pegadinha da troca de conector:** a IBFC troca um conector por outro de **sentido diferente** e pergunta se o sentido se mantém. "Estudou, **portanto** passou" (conclusão) ≠ "Estudou, **embora** passasse" (muda tudo). Identifique a relação lógica antes.
- **Macete "MAS = oposição; MAIS = quantidade":** não confunda na reescrita.
- **Pegadinha "porque/pois deslocado":** "pois" antes do verbo = causa ("Não saí, pois chovia"); depois do verbo = conclusão ("Chovia; não saí, pois.").
- **Macete da concessão:** embora / ainda que / apesar de sempre indicam que **algo aconteceu contra o esperado**.

### ✅ Questões Comentadas
**Q1.** "O agente estava cansado, _______ concluiu toda a coleta de dados." O conector que estabelece relação de **concessão/oposição** é:
A) portanto B) porque C) mas ✔️ D) logo E) por isso
**Gabarito: C.** O contraste (cansado × concluiu mesmo assim) pede conector adversativo: "mas". As demais indicam conclusão (A, D, E) ou causa (B), o que tornaria a frase incoerente.

**Q2.** Em "Apesar de não ter estudado muito, foi aprovado", a expressão "Apesar de" indica:
A) causa B) condição C) finalidade D) concessão ✔️ E) adição
**Gabarito: D.** "Apesar de" introduz uma concessão — algo ocorre **contra a expectativa** (não estudou muito, mas mesmo assim passou).

## AULA 1.12 — REDAÇÃO E REESCRITA OFICIAL

🎯 **Por que cai:** tópico **novo e específico** deste edital — 1 a 2 questões sobre clareza, padrão formal e reescrita de ofícios/comunicados.

### 📖 Resumo Teórico
- **Atributos da redação oficial** (Manual de Redação da Presidência): **clareza, concisão, objetividade, impessoalidade, formalidade/padronização e uso da norma culta**. Não se admite "encheção de linguiça", regionalismo nem gíria.
- **Impessoalidade:** o texto fala em nome do serviço público, não da pessoa. Evita-se 1ª pessoa carregada de opinião.
- **Padrão ofício** (documento unificou ofício, memorando e aviso): tipo e número, local e data, destinatário, assunto, texto (introdução, desenvolvimento, conclusão), fecho ("Atenciosamente"/"Respeitosamente"), assinatura.
- **Pronomes de tratamento na correspondência:** Vossa Senhoria (V. Sª) para autoridades em geral; concordância sempre em 3ª pessoa.
- **Reescrita:** trocar a redação mantendo o sentido, melhorando clareza/concisão. A questão dá uma frase e pede a reescrita **correta e equivalente** — atenção para não mudar o sentido nem inserir erro gramatical.
- **Vícios a evitar:** ambiguidade, prolixidade, jargão excessivo, redundância ("elo de ligação", "subir para cima").

### 🧠 Mapa Mental
```
REDAÇÃO OFICIAL
├── Atributos: clareza • concisão • objetividade
│            • impessoalidade • formalidade • norma culta
├── Padrão ofício: tipo/nº, data, destinatário, assunto,
│   texto (intro/desenv/conclusão), fecho, assinatura
├── Fecho: "Atenciosamente" (igual/inferior)
│        / "Respeitosamente" (autoridade superior)
├── Tratamento: V.Sª → 3ª pessoa
└── Reescrita: manter sentido + corrigir/clarear
    └── evitar: ambiguidade, prolixidade, redundância
```

### 🃏 Flashcards
1. P: Três atributos centrais da redação oficial? **R:** Clareza, concisão e impessoalidade (+ formalidade, objetividade, norma culta).
2. P: Fecho para autoridade superior? **R:** "Respeitosamente".
3. P: Fecho para mesma hierarquia ou inferior? **R:** "Atenciosamente".
4. P: Pode usar gíria em ofício? **R:** Não.
5. P: "Elo de ligação" é exemplo de quê? **R:** Redundância (vício a evitar).

### 🪄 Macetes & Pegadinhas IBFC
- **Macete dos fechos:** **R**espeitosamente para superio**R** (autoridade acima); **A**tenciosamente para os dem**A**is. (R com R, A com A.)
- **Pegadinha da reescrita que muda o sentido:** a alternativa parece "mais bonita", mas alterou o sentido original. A reescrita correta **mantém o significado**.
- **Macete da concisão:** se dá para cortar palavra sem perder sentido, a versão concisa é a preferida na redação oficial.

### ✅ Questões Comentadas
**Q1.** Sobre a redação oficial, assinale a alternativa **correta**:
A) Deve usar linguagem rebuscada e pessoal para demonstrar erudição.
B) A impessoalidade e a clareza são atributos essenciais. ✔️
C) O uso de gírias é permitido em comunicados internos.
D) O fecho "Respeitosamente" é usado para destinatários de hierarquia inferior.
E) A prolixidade é recomendada para detalhar o assunto.
**Gabarito: B.** Impessoalidade e clareza são pilares da redação oficial. A) contraria a impessoalidade/concisão; C) gíria é vedada; D) "Respeitosamente" é para autoridade **superior**; E) prolixidade é vício a evitar.

**Q2.** Assinale a reescrita que mantém o sentido e respeita a norma culta da frase: *"A gente vai mandar o documento pra vocês depois."*
A) Enviaremos o documento posteriormente. ✔️
B) O documento vai ser mandado pra vocês depois.
C) A gente manda o documento depois pra vocês.
D) Mandaremos pra vocês o documento aí depois.
E) O documento será mandado pra vocês mais tarde.
**Gabarito: A.** Substitui o coloquial ("a gente", "pra", "mandar") por linguagem formal e concisa ("enviaremos", "posteriormente"), preservando o sentido. As demais mantêm coloquialismos.
