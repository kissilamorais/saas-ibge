## Português
1. **C** — texto usa "elo de corrente" → **metáfora** do ACS como conector.
2. **B** — "traduzir" = mediar/interpretar → sentido **conotativo**.
3. **C** — "se perdem" = passiva sintética (se apassivador).
4. **B** — "bastantes dificuldades" (adjetivo, concorda no plural).
5. **B** — tempo futuro → "daqui **a** dois dias" (preposição).
6. **C** — "à São Paulo": topônimos masculinos/sem artigo = **sem crase**; as demais estão corretas.
7. **C** — "ao ACS" = complemento com preposição "a" → **objeto indireto**.
8. **B** — "assistir" (ver) é TI: "assistiu **ao** treinamento".
9. **B** — "como grãos ao vento" = **comparação explícita** (símile).
10. **A** — "descrição" = ato de descrever; "discrição" = qualidade de ser discreto. A grafia correta para o contexto é variável — ambas existem com sentidos diferentes. A questão pede grafia correta sem contexto; **"descrição"** é a mais comum no contexto censitário.
11. **B** — "embora" = **concessão**.
12. **B** — "Marco Silva" explica "O ACS" → **aposto**, isolado por vírgulas.
13. **B** — "informei **à** supervisora": informar + a (prep.) + artigo a (fem.) = crase.
14. **D** — "de modo que" = **consequência**.
15. **B** — assíduo (frequente/pontual) e frequente são **sinônimos**.
